# sfdc-core-system-api
Core Banking System API built on MuleSoft exposing customer, account, balance, and transaction operations. Supports seamless Salesforce integration for read/write access to banking data. Secured with Client ID &amp; Secret. Designed for system-level orchestration and data consistency.
